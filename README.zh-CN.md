# Scrollshot

> 在 Windows 上截取纵向长截图 —— 滚动、截取、拼接。

[English](./README.md)

Scrollshot 是一款命令行工具，可以在 Windows 上截取滚动长截图。选中屏幕区域后，Scrollshot 会自动向下滚动并逐帧截取，最后将所有帧拼接成一张纵向长图。

## 工作原理

1. **选择区域** —— 屏幕出现半透明遮罩层，按住鼠标左键拖拽选择截图区域，松开后在框内点击确认。
2. **自动滚动 & 截取** —— Scrollshot 在选中点发送模拟鼠标滚轮事件，通过 `xcap`（或 GDI 回退）逐帧截屏，利用计算机视觉（基于梯度的模板匹配 + 文本主体区域检测）计算相邻帧的重叠区域。
3. **拼接** —— 去除重叠部分，将所有帧合成为一张连续图像。

## 安装

```bash
cargo install --git https://github.com/xutianyi1999/scrollshot
```

或从源码构建：

```bash
git clone https://github.com/xutianyi1999/scrollshot
cd scrollshot
cargo install --path .
```

## 使用

```bash
scrollshot --output longshot.png
```

1. 出现全屏遮罩层。**按住鼠标左键拖拽** 选择截图区域。
2. 松开鼠标。**在选中区域内点击** 开始截取。
3. Scrollshot 自动向下滚动并逐帧截取。
4. 随时按 **Esc** 提前停止 —— 已截取的帧仍会保存。
5. 拼接后的图像写入 `--output` 指定的路径（默认：`scrollshot.png`）。

### 选项

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `--output <PATH>` | 输出 PNG 路径 | `scrollshot.png` |
| `--max-scrolls <N>` | 最大滚动步数 | `8000` |
| `--settle-ms <MS>` | 每次滚动后的等待时间（毫秒） | `200` |
| `--wheel-notches <N>` | 每次滚动的滚轮格数（≥1） | `4` |

### 快捷键

| 按键 | 操作 |
|------|------|
| `Esc` | 取消区域选择或提前停止截取 |

## 平台支持

**仅 Windows。** Scrollshot 依赖 Win32 API 实现 DPI 感知、GDI 截屏回退、鼠标事件模拟和选择遮罩层。需要 Windows 10 或更高版本，Rust 1.85+（edition 2024）。

## 重叠检测原理

每一帧截取后都会与前一帧对比，找出内容重叠的具体像素行。处理管线：

1. **灰度化 & 文本主体** —— 两帧统一转换灰度一次（全阶段共享），检测文本主体列区域以排除侧边栏 / UI / 滚动条干扰。
2. **特征图** —— 应用 Sobel 梯度滤波，使匹配聚焦于边缘（文字边界、UI 边框）而非纯色区域。
3. **并行模板匹配** —— 以前一帧底部为模板，在当前帧顶部区域滑动匹配（归一化交叉相关）；7 种模板高度通过 rayon 并行计算。
4. **多尺度精炼** —— 在半分辨率下粗匹配，再在全分辨率窄搜索窗内精炼。
5. **验证** —— 平方误差和验证最佳候选；检查局部 / 全局置信度裕度；5 个非重叠波段并行投票。
6. **时序平滑** —— 拼接前对单帧离群值做中值滤波抑制抖动。
